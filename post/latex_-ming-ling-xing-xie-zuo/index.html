latex_命令行写作

2021-10-28

latex的安装可以参考这篇文章最新TeXLive 环境的安装与配置，教程可以参考这个LaTeX中文学习教程

使用命令行编写latex文档

  1. 使用windows+R 打开命令提示符 输入以下命令：

    cd \
mkdir latex_text
cd latex_text
notepad text.tex  # 新建text.tex文件进入编译
# 编译内容为
	\document{article}

	\begin{document}
	Hello \Latex!

	\end{document}
# 编辑好tex文件之后保存退出

  2. 编译以及转换

    • 使用latex编译并输出pdf，输入以下命令：
      latex text.tex
# 该命令会生成text.aux，text.dvi，text.log文件
dvipdfmx text.dvi
# 该命令会将dvi文件转换为pdf文件(注意:dvi文件不会删除)
    • 使用xelatex编译并输出，输入以下命令：
      xelatex text.tex
# 生成text.aux，text.log，text.pdf等文件

  3. 使用text.pdf命令直接查看pdf文件

  4. 使用批处理文件操作将编译操作简化

    • 使用latex编译
      • 新建文本文件，将文件的后缀名改为bat
      • 使用记事本编辑输入以下命令：
        latex text.tex
dvipdfmx text.dvi
del *.dvi *.log *.aux 
pause
    • 使用xelatex编译
    • 同latex新建文件
    • 输入以下命令：
      xelatex text.tex
del *.dvi *.log *.aux 
pause
