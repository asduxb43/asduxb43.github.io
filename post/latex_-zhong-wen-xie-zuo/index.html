latex_中文写作

2021-10-28

中文写作

  1. 核心 需要完成的设置
    • latex文档使用utf-8编码
    • 在latex文档中导入了ctex宏包
    • 使用xelatex编译
  2. 使用命令行工具写作
    • 代码参见latex_命令行写作
    • 注意:
      • 保存latex文档时选择utf-8编码
      • 使用xelatex编译命令 xelatex text.tex
  3. 使用textstudio写作

    • 代码如下

        % 导言区 
  \documentclass{article} % book, report, letter
  \usepackage{ctex}

  \title{My first document}
  \author{作者}
  \date{\today}

  % 正文区（文稿区）
  \begin{document}
  \maketitle
  Hello World!
  	
  % here is my formula
  Let $f(x)$ be defined by formula
  % $之间可以写行内公式
  $$f(x)=3x^2+x-1$$ which is a polynomial of
  % $$之间可以写行间公式
  degree 2.
  \end{document}

    • 需要注意的是:需要配置textstudio

      • 修改语言：点开工具栏option、依次点击"Configure Textstudio" -> "general" -> "Language" 下拉菜单中选择"zh_CN(Chinese)",确认即可
      • 配置编译器：同样打开工具栏选项,依次点击"设置Textstudio" -> "构建" -> "默认编译器",在下拉菜单中选择xelatex，确认即可
